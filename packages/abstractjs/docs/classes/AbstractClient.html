Class AbstractClient

The AbstractClient class extends the functionality of the AbstractQueryClient.

In addition to the query functionality provided by AbstractQueryClient, AbstractClient also supports executing transactions.

Use this class when you need to perform both read and write operations.

See

  • For connecting to the blockchain. Provides access to the Abstract Name Service (AnsClient), Registry (RegistryClient), and Account Factory (FactoryClient) clients.
